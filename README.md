# AI
说明：git clone url

1，本地安装maven 版本3.3.3以上
2,进入根目录运行mvn clean package -DskipTest
3, 进入targtet目录执行java -jar ...
4,浏览器localhost:8080/ai 访问。默认账号test test

注意：默认使用oracle 数据库，启动前执行初始化数据库。
    默认接口8080 tomcat 会使用该接口。
    算法占用54321端口。
